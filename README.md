# WhatsSpam

WhatsSpam is a python script to Send Multiple WhatsApp Messages To A 'Friend' Or A Group.

### Usage : 
    python WhatsSpam.py

### Dependencies
* [Python 2/3][python]
* [Selienum][selenium]
* [Chrome][chrome]
* Matplotlib
